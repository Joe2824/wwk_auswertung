### Installieren

1. Repo clonen
   ```sh
   git clone https://github.com/github_username/repo_name.git
   ```
3. Python requirements installieren
   ```sh
   pip3 install -r requirements.txt
   ```
4. Prorgamm starten
   ```sh
   python3 run_gui.py
   ```
